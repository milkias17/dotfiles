# Dotfiles for my [Qtile](https://qtile.org) setup

## Info
----
* OS: _Arch_
* WM: [Qtile](https://github.com/qtile/qtile)
* Terminal: _kitty_
* Shell: _fish_
* Fonts: [Sora Regular](https://github.com/sora-xor/sora-font) for general text and [FiraCode](https://github.com/tonsky/FiraCode) for code
* Gtk Theme: [Nordic Darker](https://www.gnome-look.org/p/1267246/)
* File manager: _Thunar_
* Editor: _Neovim_
* Notifications: _Dunst_
* Compositor: _picom-jonaburg-git_
* Colorscheme: [Nord](https://www.nordtheme.com/)
* Login Manager: _Ligthdm-webkit2-greeter with [lightdm-webkit2-glorious theme](https://github.com/manilarome/lightdm-webkit2-theme-glorious)_
